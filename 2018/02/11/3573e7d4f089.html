git如何撤销提交

git reset –hard 这个命令可以将你已经add或commit的文件恢复到远程的版本
，但不能撤销你没有add或commit的文件即没有tack的文件和文件夹

git clean

这是一个可以操作没有被tack的文件和文件夹
git clean -n 演习 列出要被操作的文件
git clean -df 撤销未tack的文件
